API Overview

Base URL

http://localhost:8000

Authentication

Uses GP_AUTH_DATA from environment variables for Google Photos access.

Response Format

All endpoints return JSON responses unless specified otherwise.

CORS

CORS is enabled for all origins to allow browser access.

File Listing

GET /api/files/mp4 List all MP4 video files

Description

Returns a list of all MP4 video files from Google Photos with metadata including duration, file size, and TMDB information.

Response

JSON object with files array containing movie information.
